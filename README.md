# 🧹 去水印工具 Watermark Remover

基于 **OpenCV.js** 的智能图片去水印工具，纯浏览器端处理，隐私安全。

## 🎮 使用方法

1. 双击 `index.html` 打开（等待 OpenCV 引擎加载完成）
2. 上传带水印的图片
3. 用画笔涂抹水印区域（紫色高亮）
4. 选择算法和修复半径，点击「智能去除」
5. 可多次涂抹修复，满意后点击「保存图片」

## ✨ 功能特色

- 🧠 **OpenCV.js 智能修复**：TELEA / Navier-Stokes 两种专业算法
- 🔒 **隐私安全**：纯浏览器端处理，图片不上传服务器
- 🔄 **前后对比**：拖动滑块查看修复前后效果
- ↩️ **撤销功能**：最多 30 步操作撤销
- 📱 **响应式设计**：支持桌面端和移动端
- 🎨 **深色毛玻璃 UI**：Apple 风格设计

## 🚀 打开方式

直接用浏览器打开 `index.html` 即可，无需安装任何依赖。

> ⚠️ 首次加载需要从网络下载 OpenCV.js（约 8MB），请确保网络连接正常。
